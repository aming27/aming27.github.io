OSS

He de admitir que he generado muchas aberraciones y barbaridades. La mayoría de ellas las puedes encontrar en mi perfil de github. No obstante este parece un buen lugar para destacar unas pocas. Espero que alguna te pueda servir para algo...

  • vscode-solution-explorer

    Esto es una extensión para Visual Studio Code que añade un nuevo panel con un explorador de soluciones. Está basado en el "Solution Explorer" que podemos encontrar en Visual Studio Enterprise, Professional o Community, pero su catálogo de características no es tan completo. Puede resultarte muy útil para el desarrollo con dotnet core o .Net 5.0.

    Búscalo en el panel de extensiones de Visual Studio Code para instalarlo.

  • Xunit.Assume

    Para ayudar con la documentación del código usando unit tests. Para muchos Assume es la cuarta A de las pruebas unitarias (Arrange, Act, Assert). La idea es añadir un momento en el que comprobamos si el estado actual del sistema es susceptible de ser probado. Assume dejará implícito el cuándo y el por qué estamos saltando la ejecución de un si un test.

    Puedes descargarlo como un paquete de NuGet llamado "xUnit.Assume".

  • types-vue

    Es un paquete de npm que te ayuda a trabajar con TypeScript en Vue.js. Imita la notación que podemos encontrar a trabajar con plataformas como angular. Añade soporte para el comportamiento básico de las versiones v1 y v2 de Vue y la librería Vuex.

    Puedes descargarlo con un "npm i types-vue".
